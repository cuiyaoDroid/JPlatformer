==================
1) ÜBER DAS SPIEL
==================

JPlatformer ist ein klassisches Jump-'n'-Run-Spiel. Ziel ist es, die Spielfigur
unbeschadet zum Levelende zu steuern und dabei allerhand Hindernissen
auszuweichen, denn jede Kollision mit Gegnern und Fallen endet tödlich. Dabei
gilt es, Höchstpunktzahlen und Bestzeiten zu erreichen.

Im Programm integriert ist der Karteneditor, ein Kernelement des Spiels, welcher
das Erstellen und Testen eigener Welten ermöglicht.

JPlatformer ist in Java geschrieben und läuft somit auf allen gängigen
Plattformen. Um das Spiel zu starten, wird Java 8 oder höher benötigt.

=============
2) STEUERUNG
=============

Die Steuerung im Spiel:

- Nach links    -> Pfeiltaste links
- Nach rechts   -> Pfeiltaste rechts
- Springen      -> Leertaste
- Benutzen      -> Pfeiltaste oben
- Pause-Menü    -> Escape, P
- Kamera zoomen -> Mausrad (scrollen), Plus / Minus

Die Steuerung im Editor:

- Seitenleiste öffnen / schließen -> Strg + H
- gewähltes Element platzieren    -> Linke Maustaste
- markierte Elemente entfernen    -> Rechte Maustaste
- Objekte frei platzieren         -> Alt
- Objekte markieren               -> Strg + Linke Maustaste
- Kamera bewegen                  -> Mausrad (gedrückt halten), Q
- Kamera zoomen                   -> Mausrad (scrollen), Plus / Minus
- Wasserhöhe festlegen            -> W
- Spiel pausieren                 -> Escape, P
